Protein CrossLinks Explorer.

Usage:
  crosslinx.jar [all [--list-only]] [--no-gui] [--pdb-dir <PDIR>] [--home-dir <HDIR>]
  crosslinx.jar from-file <FILE> [--no-gui] [--pdb-dir <PDIR>] [--home-dir <HDIR>]
  crosslinx.jar from-list <LIST> [--no-gui] [--pdb-dir <PDIR>] [--home-dir <HDIR>]
  crosslinx.jar (-h | --help)
  crosslinx.jar --version




Options:
  -h --help           Show this screen.
  --version           Show version.
  --list-only         Generate a list of files from a local directory and exit. [default: false]
  --pdb-dir=<PDIR>    Local PDB of mmCIF store (backup) folder. If it is not
                      given, it will be taken from the settings file.           [default setting is current folder]
  --home-dir=<HDIR>   The local work and temp folder.  If it is not given, it 
                      will be taken from the settings file.                     [default setting is current folder]
  all                 Explore all files in local PDB store. This is the default 
                      behaviour if no commands are given.
  from-file=<FILE>    File containing list of PDB IDs to parse, in the format
                      generated by the --list-only option. If some PDB IDs are 
                      not in the local directory, will try to fetch them from wwPDB.
  from-list=<LIST>    Semicolon-separated list of PDB IDs. Can be on multiple 
                      lines. Lines starting with the `#` sign are ignored.
  --no-gui            Do not show GUI (use command line options only).          [default: false]
